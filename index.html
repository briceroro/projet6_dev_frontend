Regles

  • 3 déplacements max par tour
  • Vous pouvez passer votre tour avant les 3 déplacement

Déplacement

  • Haut : touche z
  • Droite : touche d
  • Bas : touche s
  • Gauche : touche q
  • Passer son tour : x

Armes

  • Drapeau : force + 5
  • Vin : force + 10
  • Sabre : force + 15
  • Canon : force + 20

Mode combat

  • attaquer : touche z
  • se défendre : touche s
  • Défendre réduit de 50% les dégats.
